Student Performance Analysis using Python

Objective:
Analyze student academic performance data and identify key factors affecting outcomes.

Tools Used:
Python, Pandas, NumPy, Matplotlib

Key Work:
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Correlation analysis
- Feature engineering (total score)
- Visualization of insights

Key Insights:
- Students completing test preparation performed better overall
- Strong correlation between subject scores
- Aggregate performance metrics help identify top students

Conclusion:
This project demonstrates practical data analysis and problem-solving skills using Python.
